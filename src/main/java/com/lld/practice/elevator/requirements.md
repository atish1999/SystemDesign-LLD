
###### REQUIREMENTS
<hr/>

- will be having 3 elevators and 10 floors
- there will be hall call (Up and down) from the floor by user.
  - System will decide which elevator dispatch
- user can select destination from the inside elevator
- user can select multiple destinations in the elevator 
- Elevator stops comes in 2 steps 
  - select from the floor by using hall call direction included UP or DOWN
  - from inside the elevator destination can be chosen without direction
- there will be a simulation software which will call stop()
- Invalid request should be rejected
- for the same floor request will be no ops
